Intranet Projekt

Laufend

Entwicklung eines schulischen Intranets, das ausschließlich via VPN zugänglich ist. Das Netzwerk umfasst einen Proxmox Cluster bestehend aus 2 Nodes sowie einen weiteren Cluster aus 8 Nodes. Schüler erhalten VPN-Tokens, um auf Dienste wie NextCloud, Emby, Adguard, 3cx Telefonsystem, Minecraft Server und weitere interne Dienste zuzugreifen. Ein interner DNS-Server ermöglicht die Erstellung eigener Domains.

ESP Notbremse

Laufend

Von einem Freund erhielt ich eine Notbremse einer U-Bahn, die ich an einen ESP32 anschloss. Bei Betätigung zeigt ein I2C Display einen Alarm an und sendet Benachrichtigungen an Mobiltelefone. Das Projekt wird um eine weitere Notbremse mit Lautsprecher und Mikrofon erweitert, um eine Freisprecheinrichtung vom Serverraum in mein Zimmer zu realisieren.
