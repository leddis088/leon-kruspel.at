ESP Emergency Brake

Ongoing
Project Overview

Project Description

I received a subway emergency brake from a friend, which I connected to an ESP32. When activated, an I2C display shows an alarm and sends notifications to mobile phones. The project is being expanded with another emergency brake with speaker and microphone to create an intercom from the server room to my room.

Technical Details

  • ESP32 microcontroller with WiFi capabilities
  • I2C Display for status information
  • Push notifications via Pushover API
  • Discord integration for alerts
  • Custom 3D printed case

Future Plans

  • Adding speaker and microphone for intercom functionality
  • Implementing two-way communication
  • Creating a web interface for configuration
